# Teer Scraper (Next.js)

This is a minimal Next.js starter that scrapes https://teertooday.com/ for Shillong Teer results.

## Features
- `/api/scrape` server API that scrapes the homepage and returns JSON.
- Simple homepage that fetches the API and displays results.
- Ready to push to GitHub and deploy on Vercel.

## Run locally
1. Install dependencies: `npm install`
2. Run dev: `npm run dev`
3. Open http://localhost:3000

## Deploying
- Push to a GitHub repo and connect to Vercel (recommended).
- Or upload the repo to Vercel during project creation.

## Notes
- Respect the target site's terms and robots.txt. Don't overload their servers.
